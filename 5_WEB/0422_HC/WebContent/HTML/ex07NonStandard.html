웹 표준을 지키지 않은 상태

이제 웹표준의 역할은 아주 중요합니다. 

  선택자 {  태그 속성 : 값 ; }
  선택자, 선택자2, 선택자3,... {  태그 속성 : 값;  태그 속성 : 값;  }
  선택자 서브엘리먼트 {  태그 속성 : 값 ; }
   p em { 속성:값, 속성:값,... }
   div span { 속성:값, 속성:값,... }

  .className{ 속성:값, 속성:값,... }
  ex) .red { color: red; } 
  
   선택자.클래스이름 { 속성:값, 속성:값,... }
  ex) em.b { color: blue }

   #idName { 속성:값, 속성:값,... }
   div#id  { 속성:값, 속성:값,... }
   #header  { 속성:값, 속성:값,... }
